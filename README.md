## 游눫 Community & Discussions

Since the actual codebases are on GitLab, we use **GitHub Discussions** in this repository for all community interactions. 

### How to use Discussions:
* **[Announcements](https://github.com/MelodicAlbuild/hytale/discussions/categories/announcements):** Release Notes, Breaking Announcements, and more!
* **[General Chat](https://github.com/MelodicAlbuild/hytale/discussions/categories/general-chat):** For casual conversation about the projects.
* **[Feature Requests](https://github.com/MelodicAlbuild/hytale/discussions/categories/feature-requests):** Have an idea for the server or API? Post it here.
* **[Q&A](https://github.com/MelodicAlbuild/hytale/discussions/categories/q-a):** Stuck on implementation or have a question about the roadmap? Ask here.
* **[Show and Tell](https://github.com/MelodicAlbuild/hytale/discussions/categories/show-and-tell):** Show off what you've built using these tools.
* **[Polls](https://github.com/MelodicAlbuild/hytale/discussions/categories/polls):** If I ever want to hear the communities opinion, I will ask here!

## 游냍 Reporting Bugs

Please do not open GitHub Issues in this repository unless it is related to this README itself. 

* **To report a bug:** Please open a Discussion thread under the "[Bug Reports](https://github.com/MelodicAlbuild/hytale/discussions/categories/bug-reports)" category.
* Include the project name, version, and reproduction steps.
